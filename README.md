# Uttara-Times

**Uttara-Times** is a dynamic, web-based news portal built with **PHP, MySQL, and Bootstrap 5**. The platform allows users to browse, filter, and read articles by category, tag, or author. It features a subscription system for premium content, advertisement management, and user authentication.

## Features

- User authentication and role management (Admin, Journalist, Reader)
- Article management with tags, categories, and image support
- Advanced filtering and sorting by author, tag, and date
- Subscription system with plan management and expiration handling
- Advertisement management (top banner, sidebar, and pop-ups)
- Responsive UI using **Bootstrap 5** for mobile and desktop
- Logging system for user activity and subscription status

## Technologies Used

- PHP 8+
- MySQL / MariaDB
- HTML5, CSS3, JavaScript, Bootstrap 5
- Font Awesome icons
